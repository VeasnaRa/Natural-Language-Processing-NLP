# NLP Project: Analysis of 'The Little Prince' Chapter I

## Project Objective
This project focuses on the design and implementation of a comprehensive Natural Language Processing (NLP) pipeline for the analysis of Chapter I of Antoine de Saint-Exupéry's novel, "The Little Prince." The goal is to explore various NLP techniques and models to understand and process textual data.

## Technologies and Domains Explored

*   **Word Embeddings (NLTK & Gensim):** Utilized NLTK for tokenization and Gensim for training Word2Vec models (CBOW and Skip-gram). Visualized 2D embeddings using t-SNE.
*   **Deep Neural Networks (TensorFlow/Keras):** Developed and trained a dense neural network for contextual word prediction from embeddings. Integrated and evaluated a multi-head self-attention mechanism to compare performance.
*   **Transformer Models (HuggingFace):** Applied the CamemBERT model for Masked Language Modeling (MLM) and extracted token representations.
*   **Large Language Models (LLM):** Integrated the Google Gemini API for automatic text summarization.

## Developed Skills

*   Text data preprocessing (tokenization).
*   Construction and evaluation of machine learning models (Word2Vec, Neural Networks, Transformers).
*   Use of advanced NLP libraries (NLTK, Gensim, TensorFlow/Keras, HuggingFace Transformers).
*   Interaction with LLM APIs.
